# Cursor Commands

Các slash commands (gõ `/` trong Cursor) để tự động hoá workflow cho repo này:

- `sql-schema` – thiết kế & migrate Prisma schema
- `architecture-web` – chuẩn hoá kiến trúc hệ thống & docs
- `create-feature-frontend` – tạo tính năng UI theo pattern sẵn có
- `scaffold-module` – scaffold module dashboard nhanh bằng script
- `deploy-cloud-run` – deploy Cloud Run chuẩn
- `oauth-google` – cấu hình Google OAuth (NextAuth)
- `triage-logs` – đọc logs & fix production bugs nhanh
- `perf-audit` – tối ưu performance có hệ thống
- `agent-review` – review thay đổi trước khi deploy
